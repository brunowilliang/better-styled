# Changelog

## 1.0.0-beta.4

- Initial beta release
- `styled()` - Create components with variants
- `createStyledContext()` - Share variants between parent and child
- `withSlots()` - Compound components with dot notation
- `cn()` - Class merging with tailwind-merge
- `mergeFinalProps()` - Props merging with function composition
- `mergeStyles()` - Style object merging
- React and React Native support
